    Jim Brady • 4 months ago

    I've followed entanglement since 1977, quantum computing since it arrived and continually wondered if there were enough applications to justify building other than a toy machine. My current take is a Quantum Computer (QC) does not Add, Subtract, etc. very well. It will not replace von Neumann machines. The 'killer apps' are code cracking (which I really don't want), and optimal route finding, where we already have pretty good approximation algorithms. Someone needs to describe what it can do in English, how long it takes to set up a computation, and how long to do it. Until then QC is just a source of not too comprehensible papers.

    My guess based on the incoherence in the literature is it will go the way of magnetic bubbles, tagged memory architectures, holographic storage, CCD storage, etc. The lesson they taught was it take a LOT of leverage to beat an entrenched technology. A LOT!
